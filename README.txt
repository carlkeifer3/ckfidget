ckFidget.py

part of my custom 3d toolchain. this tool allows you to 
collect single attributes from multiple objects in one place, 
and work with them by adjusting their values as a group or 
individually by increment or percentage.


   currently supported:
      maya2012
      
   to install:
      place in scripts directory
      
      import ckFidget as fdg
      
      fdg.ckFidgetInit()
